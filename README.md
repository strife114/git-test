# PyProj 测试项目

这是一个 Python 练手测试项目，用于学习和实验 Git 版本控制。包含以下模块：

## 文件说明

| 文件 | 功能 |
|------|------|
| `stock_growth.py` | 股票增长计算 |
| `snake_game.py` | 贪吃蛇小游戏 |
| `line_chart.py` | 折线图绘制 |
| `time_test.py` | 显示当前日期时间 |
| `test1.py` | 综合测试脚本 |
| `test_new.py` | 新测试用例 |
| `test_demo.py` | 演示测试 |

## Git 学习内容

本项目用于练习 Git 常用操作：
- 初始化仓库与推送 GitHub
- 分支创建、切换、合并（Fast-forward / 三路合并）
- 解决合并冲突
- 版本回退（reset / revert）
- 工作树管理
- 强制推送与历史重写
